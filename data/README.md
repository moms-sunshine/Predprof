# Данные

- **Обучающий/валидационный набор:** скачать `Data.npz` по ссылке https://disk.yandex.ru/d/BA4oJb0BwaABxg и положить в эту папку (или указать путь в скрипте обучения).
- **Тестовый набор:** по ссылке https://disk.yandex.ru/d/39RhMM5NiJvgcg (пароль выдаётся жюри). После загрузки через приложение результаты отображаются в аналитике.

В `.npz`:
- `train_x` — wav-файлы (обучение), `train_y` — повреждённые метки (строки)
- `valid_x` — wav (валидация), `valid_y` — метки
- В тестовом: `test_x`, `test_y`

Метки восстанавливаются до целых 0, 1, 2, ... в модуле `restore_labels`.
